# Mathhammer Matrices

Interactive Warhammer 40,000 damage-comparison tools. Each is a self-contained HTML page: pick a target, toggle buffs/stratagems/marks/weapons, and see expected damage (models killed or wounds dealt) recompute live.

Live: **https://sean-iam.github.io/40k-matrices/**

- `index.html` — gallery / launcher
- `defiler-damage-matrix.html` — Defiler, Soulforged vs Pactbound (shooting)
- `incubi-damage-matrix.html` — Incubi + Archon vs + Lady Malys, Skysplinter Assault (melee)

Expected-value math; assumptions are documented in each tool. Not affiliated with Games Workshop.
